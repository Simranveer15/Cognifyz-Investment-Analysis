# Investment Behavior Analysis Dashboard 📊
**Cognifyz Technologies Data Visualization Internship - Final Project**

## Project Overview
This project involves a comprehensive analysis of investment preferences based on demographic data. The goal was to transform raw survey data into an interactive Excel dashboard that reveals how age, gender, and financial objectives influence investment choices.

## Key Features
- **Interactive Slicers:** Dynamic filtering by Gender and Age groups.
- **Advanced Visualizations:** Includes a **Correlation Matrix**, **Radar Chart** for return expectations, and **Bubble Charts** for age-impact analysis.
- **Professional UI:** Clean, grid-free design focused on user experience.

## Business Insights
- **Top Avenue:** Mutual Funds are the most preferred investment vehicle (45% share).
- **Growth Focus:** 65% of participants prioritize **Capital Appreciation** over liquidity.
- **Expectation Gap:** Most investors expect **20-30% returns**, highlighting an aggressive market sentiment.
- **Demographic Trend:** The **26-30 age bracket** shows the highest engagement in active investing.
